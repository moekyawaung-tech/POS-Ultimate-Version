# POS Ultimate System

## English
A ready-made ultimate POS project for mini markets and shops with admin CRUD, stock management, discount system, cash received/change, sales report, export CSV, Chart.js chart, localStorage, bilingual support, and MMK/Baht currency format.

## မြန်မာ
ဒီ project သည် မီနီမားကတ်နှင့် စတိုးဆိုင်များအတွက် အသင့်သုံးနိုင်သော POS Ultimate System ဖြစ်ပြီး အောက်ပါ features များ ပါဝင်ပါသည်။

### Features
- Login Page
- Admin Product CRUD
- Product Search
- Add to Cart
- Stock Management
- Discount System
- Cash Received / Change
- Receipt Print
- Sales Report
- Export CSV
- Chart.js Sales Chart
- LocalStorage Save
- Burmese / English
- MMK / Baht
